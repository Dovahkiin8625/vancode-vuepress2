## 数字

- 整型(int) - python3 整型是没有限制大小的，可以当作 Long 类型使用，所以 python3 没有 python2 的 Long 类型
- 浮点型(float) - 由整数与小数组成，也可用科学计数法表示（2.5e2 = 2.5x102 = 250）
- 复数(complex) -实数和虚数构成，可用a + bj 或complex(a,b)表示，a和 b都是浮点型


```python
a = 1
b = 1.0
c = 1+2j
d = complex(1, 2)

type(d)
```
