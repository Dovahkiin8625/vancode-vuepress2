## format
[回到顶部](#字符串)

python2.6 开始，新增了一种格式化字符串的函数 format()，它增强了字符串格式化的功能。<br>
其使用 {} 做占位符，可以通过不同的数据类型传入数据

- 传入方式


```python
# 最常用的传入方式

print("{}, {}".format("a", "b"))
print("{0}, {1}, {0}".format("a", "b"))
print("{p1}, {p2}, {p1}".format(p1="a", p2="b"))
```

    a, b
    a, b, a
    a, b, a



```python
# 通过列表传入数据

lis = ['a','b']
print('{names[0]}, {names[1]}'.format(names=lis))
print('{0[0]}, {0[1]}'.format(lis))
```

    a, b
    a, b



```python
# 通过字典传入数据

dic = {"a": "a", "b": "b"}
print("{names[a]}, {names[b]}".format(names=dic))
```




    'a, b'




```python
# 通过形参传入数据

args = ["0", "1"] 
kwargs = {"a": "0", "b": "1"} 
print("{}, {}, {a}, {b}".format(*args, **kwargs))
```

    0, 1, 0, 1


- 其他功能


```python
print("{:.2f}".format(3.1415926))
print("{:.1%}".format(0.25))
```

    3.14
    25.0%



```python
print("{:-^10}".format("a"))
print("{:-<10}".format("a"))
print("{:->10}".format("a"))
```

    ---------a
